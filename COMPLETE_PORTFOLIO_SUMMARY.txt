📞 QUICK SUPPORT REFERENCE
================================================================================

PROBLEM: Links don't work
SOLUTION: 
  • Ensure all 5 files in same folder
  • Check filenames match exactly (case-sensitive)
  • Test locally before deploying
  • Hard refresh browser (Ctrl+Shift+R)

PROBLEM: Website looks broken
SOLUTION:
  • Clear browser cache
  • Check if fonts loaded from Google
  • Look at browser console for errors
  • Try different browser

PROBLEM: Mobile looks wrong
SOLUTION:
  • Test on actual mobile device
  • Check viewport meta tag present
  • Adjust CSS media queries
  • Test different screen sizes

PROBLEM: Can't deploy to GitHub
SOLUTION:
  • Create repo first
  • Upload files through web interface
  • Or use git commands from guide
  • Enable GitHub Pages in settings
