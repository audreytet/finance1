Stock Price Prediction Using ARIMA

Time Series Forecasting for Ghana Stock Exchange

Project Overview

This project implements an advanced ARIMA (AutoRegressive Integrated Moving Average) model to forecast stock prices on the Ghana Stock Exchange. The analysis demonstrates statistical forecasting techniques, parameter optimization, and model validation on real financial time series data.

Methodology

1. Data Preparation & Exploration

The analysis begins with loading Ghana Stock Exchange historical daily closing prices. Data is cleaned to handle missing values using forward-fill methods, and basic exploratory statistics are computed.

2. Stationarity Testing

The Augmented Dickey-Fuller (ADF) test is applied to determine if the time series is stationary. Non-stationary data is differenced (d=1) to achieve stationarity, which is a requirement for ARIMA modeling.

# ADF Test Results:
# Original Series: Non-stationary (p-value > 0.05)
# First-Differenced: Stationary (p-value < 0.05) ✓

3. Parameter Identification

Autocorrelation Function (ACF) and Partial Autocorrelation Function (PACF) plots guide the selection of p and q parameters. The integration order (d) is determined from stationarity testing.

4. Model Selection via Grid Search

A grid search evaluates multiple ARIMA(p,d,q) combinations, selecting the model with the lowest AIC (Akaike Information Criterion) score. The optimal model identified was ARIMA(2,1,2).

5. Model Validation

The model is validated on a test set (20% of data) using metrics: RMSE, MAE, MAPE, and directional accuracy. The model achieved 72% directional accuracy, outperforming benchmark techniques.

6. Forecasting & Confidence Intervals

The fitted model generates future forecasts with 95% confidence intervals, providing both point estimates and uncertainty bounds for decision-making.

Results & Insights

Performance Metrics

ARIMA(2,1,2) Model Results:

  • RMSE: 2.08 (Root Mean Squared Error)
  • MAE: 1.65 (Mean Absolute Error)
  • MAPE: 2.3% (Mean Absolute Percentage Error - excellent for forecasting)
  • Directional Accuracy: 72% (Correct price direction prediction)

Benchmark Comparison

The ARIMA model outperformed traditional forecasting methods:

  • Simple Moving Average: RMSE 3.45, DA 58%
  • Exponential Smoothing: RMSE 2.95, DA 65%
  • Linear Regression: RMSE 4.12, DA 52%

Key Findings

  • GSE shows moderate upward trend with seasonal volatility
  • ARIMA is most effective for 1-4 week price forecasts
  • Model captures temporal dependencies in price movements
  • Confidence bands widen beyond 2-week horizon, indicating increasing uncertainty
