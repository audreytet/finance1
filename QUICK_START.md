# Portfolio Quick Start Guide

## 🚀 Get Started in 3 Steps

### Step 1: Push to GitHub (5 minutes)

```bash
# Navigate to portfolio directory
cd benaifa-portfolio

# Initialize git
git init

# Configure git
git config user.name "Benaifa Addo-Bartels"
git config user.email "benaifa95@gmail.com"

# Add all files
git add .

# Commit
git commit -m "Initial portfolio commit: Financial Analyst Projects"

# Create GitHub repo at github.com, then:
git remote add origin https://github.com/YOUR_USERNAME/benaifa-portfolio.git
git branch -M main
git push -u origin main
```

### Step 2: Share Your Portfolio

Copy and share this link:
```
https://github.com/YOUR_USERNAME/benaifa-portfolio
```

Add to your:
- ✅ LinkedIn profile (Website section)
- ✅ Resume/CV
- ✅ Email signature
- ✅ Job applications
- ✅ LinkedIn headline

### Step 3: Customize

Edit the following files to personalize:

1. **README.md** - Update your contact info
2. **GITHUB_SETUP_GUIDE.md** - Follow installation instructions
3. **Projects** - Add your own data and results
